## 2.4.0 codename "Windhoek" (current version)
- Added hotkeys for copying, inserting, cutting, and deleting
- Added support for switching color themes and languages
- CTkMessagebox was forked for Brushshe's needs
- The structure has been reorganized
- The interface and logic are separated

## 2.3.0 codename "Vientiane"
- Added select by color features
- Added support for more formats, including one-sided PDF support
- Stickers and frames are now CC0 1.0-marked
- Added support for loading stickers from URLs
- Added support for resizing copied or cut areas
- You can now resize real-size stickers along with copied areas
- The program exit notification now appears only when there are unsaved changes
- The "File" menu has been organized more logically
- Updated color theme - removed blue elements
- Added support for displaying selections as ants
- Added gradient fill
- Effects from a separate window have been moved to the top panel of the main window
- Added brightness and contrast effects
- Added autosave
- Now scrolling normally in Linux windows (previously only in the gallery)
- New screenshot utility that allows you to highlight directly on the screen
- Added a button to select a random color
